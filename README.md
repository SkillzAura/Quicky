# Quicky

Quicky is a **Quick Settings** project intended to provide a fast and simple way to access common settings and actions.

## Description

This repository currently contains the initial project setup and documentation.  
The goal is to keep Quicky lightweight, easy to understand, and straightforward to extend.

## Project Status

🚧 Early stage / bootstrap setup.

## Repository Structure

- `README.md` — project overview and usage notes
- `LICENSE` — project license

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/SkillzAura/Quicky.git
   ```
2. Open the project folder:
   ```bash
   cd Quicky
   ```
3. Follow upcoming setup instructions as the project evolves.
